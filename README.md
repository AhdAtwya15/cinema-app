# 🎬 Cinema-Booking-App

A modern, full-featured cinema booking platform built with **React 19**, **Vite**, and **Tailwind CSS 4**. This application provides a seamless experience for users to browse movies, select seats, and manage bookings, while providing admins with a robust dashboard for movie and booking management.
---
## ✨ Key Features

### 👤 User Interface
- **🎬 Movie Highlights**: Explore the latest releases and trending movies with dynamic hero sections and news updates.
- **🔍 Movie Discovery**: Browse detailed movie information, including trailers, genres, and descriptions.
- **🪑 Interactive Seat Selection**: A visually intuitive seat selection system for choosing the perfect spot.
- **💳 Secure Payments**: Integrated payment flow (via Stripe) for a smooth booking experience.
- **📝 User Accounts**: Authentication system for users to view their booking history.
- **📱 Fully Responsive**: Optimized for all devices, from desktops to mobile phones.

### 🛠️ Admin Dashboard
- **📊 Analytics Dashboard**: View summaries of bookings and site performance.
- **🎞️ Movie Management**: Add, edit, or remove movies from the listing.
- **📅 Booking Overview**: Manage all user bookings and track seat availability.
- **🛡️ Protected Routes**: Secure admin area accessible only to authorized personnel.

---

## 🚀 Tech Stack

- **Framework**: [React 19] + [Vite]
- **Styling**: [Tailwind CSS 4] + [Framer Motion] (Animations)
- **UI Components**: [Shadcn UI] + [Radix UI]
- **State Management**: [TanStack Query] (React Query)
- **Routing**: [React Router 7]
- **Forms**: [React Hook Form] + [Yup]
- **Icons**: [Lucide React]
